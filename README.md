# CrisisMesh

## Community-Driven Emergency Response Platform

CrisisMesh is a web platform designed to help communities respond faster during emergencies. It connects people who **need help** with nearby **volunteers and resources** in real time.

During disasters like floods, earthquakes, accidents, or large emergencies, communication becomes difficult and coordination becomes slow. CrisisMesh provides a **simple and fast system** where users can report urgent needs and volunteers can offer help.

The platform creates a **community-powered crisis response network** to improve coordination and reduce response time.

---

## Problem

During emergencies:

- People struggle to communicate urgent needs
- Volunteers and NGOs don't always know where help is required
- Information gets scattered across social media
- Coordination between helpers and victims becomes inefficient

This leads to **delayed response and unmanaged resources**.

---

## Solution

CrisisMesh connects **urgent requests** with **available help** on a single platform.

Users can:

- Report emergency needs
- Offer available resources
- View ongoing crisis reports
- Coordinate help quickly

This creates a **real-time community response system**.

---

## Key Features

### Report Needs
Users can post urgent requirements such as:

- Food
- Medical aid
- Shelter
- Rescue assistance

These requests become visible to nearby volunteers and organizations.

### Offer Resources
Volunteers, NGOs, and citizens can register available resources like:

- Food supplies
- Medicine
- Transportation
- Shelter
- Rescue support

### Smart Matching
The platform connects reported needs with available resources to help responders identify where assistance is required.

### Requests and Offers Dashboard
Users can browse all current requests and available help in one place.
